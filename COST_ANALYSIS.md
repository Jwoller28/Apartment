# Cost Analysis

Expected MVP cost: **$0**.

## GitHub Pages

GitHub Pages is free for this static React/Vite frontend when hosted from the repository.

## Supabase

Supabase Free tier should be enough for this private toy app:

- only 3 trusted users
- one shared apartment document
- small `furniture_items` table
- light realtime presence and item updates

The app uses the public anon key with permissive Row Level Security policies meant only for a private link shared with Jordan, Camila, and Ari.

## Domain

Using the default `github.io` URL is free. A custom domain is optional and only costs money if you decide to buy one.

## Apple Devices

The app runs in iPhone and iPad browsers, so there is no App Store fee and no native iOS build cost.

## Total

For the MVP as described, the likely total is **$0**.
